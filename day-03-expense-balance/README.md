# Balance Tracker (Practice Project)

## What it does
Balance Tracker app that tracks your **balance**, **income**, and **expense**. It also lets you **delete a transaction** when a user makes a mistake. The balance updates based on the total income and total expense.

## Why I built it
I wanted to practice **CRUD operations** in React using **useState** and **props**, while also making **reusable components**.

## What I learnt
I learned how to do **CRUD operations** in React, how to use **props**, and how to create **reusable components**.

## Tech used
- React
- Tailwind CSS
